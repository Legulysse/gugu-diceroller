## Quick Overview

This is a simple dice roller intended for tabletop rpg sessions.  
I couldn't find one online that was enjoyable to use and decided to make my own.  

It features all standard die types, and you can use ctrl+v after a roll to paste your results.  

You can download the latest release here :  
https://github.com/Legulysse/gugu-diceroller/releases/download/v1.1/DiceRoller-1_1.zip
